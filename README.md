# Swap-upgrade (Core) — Лабораторна робота

Виконано оновлення архітектури проєкту: впровадження власних колекцій та механізмів сортування.

### 🛠 Реалізований функціонал:
* **Кастомна колекція:** Створено клас на основі масиву `Model[] items` з методами `Add`, `RemoveAt`, `GetAt`, `SetAt` та властивістю `Count`.
* **Ітерація:** Реалізовано інтерфейс `IEnumerable` та **власний `IEnumerator`**. Перебір у меню реалізовано через `it.MoveNext()`.
* **Сортування (Natural sort):** У головній моделі додано `IComparable` для стандартного сортування.
* **Сортування (Alternative sort):** Створено окремий клас у папці `Comparers` через інтерфейс `IComparer`.
* **Меню:** Додано пункт **Stats** для відображення статистики проєкту та функціонал сортування списку.
